# Movie Rental App

## Project Overview

The Movie Rental App is a React Native application that allows users to browse and rent movies using data from The Movie Database (TMDB) API.

## Features

- Three Screens:
  Home Screen: Search for movies via TMDB API and browse a scrollable list of movie cards.
  Rented Screen: View a list of rented movies in a scrollable layout.
  Watch Screen: Watch a placeholder video and mark movies as watched to remove them from the rented list.

- Two Dialogs: Confirmation dialogs for renting and returning movies.

- Global Theme: Consistent styling throughout the app.

- Custom Font: Enhances the app's visual appeal.

- Context API: Two context files for managing global state.

- Persistent Storage: Uses AsyncStorage to save user preferences and rental history.

- Navigation: Implemented using React Navigation Stack for smooth transitions.

- UI Components: Leverages the React Native Elements UI Kit for a polished design.
